Repository of packages used by my esphome devices. 
Provided YAML files are not sufficient for full operation of sensor but are meant to speed up the process of conifgs, to make most of them reusable. 

Sensor specific requirements: 
- m5_env4 - m5stack env. IV sensor (sht40 + BMP280) requires grove-i2c package import
- bh1750 - requires grove-i2c package import
